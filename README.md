# Expense Tracker
Expense Tracker using Python integrating OpenAI

## Learning Python with freeCodeCamp
The code is from [freeCodeCamp](https://www.freecodecamp.org/news/build-smart-expense-tracker-with-python-and-llms/#heading-how-to-set-up-the-expense-data). It uses OpenAI to predict the category based on the description given on the form for adding expenses. The application shows the visuals for the expenses for meaningful interpretation.
